# Smart Meter Integration

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-41BDF5.svg)](https://github.com/hacs/integration)

## Features

This integration provides comprehensive monitoring of Smart Meter Gateway devices, including:

- **Electricity monitoring** - consumption, return, power per phase
- **Gas monitoring** - total and hourly consumption
- **Network status** - WiFi signal strength
- **Device information** - firmware version and update status

## Quick Start

1. Install through HACS
2. Restart Home Assistant
3. Add integration via UI (Settings → Integrations → Add → Smart Meter)
4. Enter your Smart Meter gateway IP and port
